How to Get to Vim and Navigate It

  1. Open your computer's terminal
  2. Navigate to your destop directory, then to what bigger file contains the file you want to access
  3. Type: 'Vim file name'
  4. Here you are in the vim part of your terminal and you are able to edit your file
  5. To enable edits, you need to type: 'i'
  6. Then, to get out of edit mode, you need to press: 'esc'
  7. If you want to leave vim, you must be out of edit mode. Then, you need to tpye: ':wq'
  8. If you are presented with something along the lines of: Error 55, then you might need to use another method of closing Vim. One of these other methods is typing: ':q '
